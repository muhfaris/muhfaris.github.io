#How To Install Git

## Linux
- `sudo apt install git`
- 'sudo apt-get install git'
- `yum install git`
- 'sudo pacman -S git'

	 Choose according to your operating system

## Windows
- open in your browser 'https://git-scm.com/'
- choose 64bit / 32bit
